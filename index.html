BioDB

Metabolic Database Integration Pipeline

The pipeline creates compound and reaction sets from metabolic databases. The compilation method fuses theses sets into single records to display an unified database. The web viewer allows to inspect the final solution. Since the viewer does not rely on any server side processing the entire data is loaded into the browser. Large sets may consume a lot of browser memory, the viewer is only for debugging and demonstration purposes.
